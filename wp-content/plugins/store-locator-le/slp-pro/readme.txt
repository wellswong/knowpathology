A premium add-on for the Store Locator Plus location management system for WordPress.

== Description ==

This plugin will help site admins manage large listings of locations with tools for processing location data en mass.

== Installation ==

= Requirements =

* Store Locator Plus: 3.9.7+
* Wordpress: 3.5.1+
* PHP: 5.2.4+

= Install After SLP =

1. Go fetch and install Store Locator Plus version 3.9.7 or higher.
2. Purchase this plugin from CSA to get the latest .zip file.
3. Go to plugins/add new.
4. Select upload.
5. Upload the zip file.
